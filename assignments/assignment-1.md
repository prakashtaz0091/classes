### **Python Practice Questions with Hints/Algorithm**

1. **Add two numbers**
   **Hint:**

   - Input two numbers.
   - Use `+` operator.
   - Print the result.

2. **Subtract two numbers**
   **Hint:**

   - Input two numbers.
   - Use `-` operator.
   - Print the result.

3. **Multiply two numbers**
   **Hint:**

   - Input two numbers.
   - Use `*` operator.
   - Print the result.

4. **Divide two numbers**
   **Hint:**

   - Input two numbers.
   - Use `/` operator.
   - Print the result.

5. **Remainder of two numbers**
   **Hint:**

   - Input two numbers.
   - Use `%` operator.
   - Print the remainder.

6. **Square of a number**
   **Hint:**

   - Input a number.
   - Multiply the number by itself or use `**2`.
   - Print the result.

7. **Cube of a number**
   **Hint:**

   - Input a number.
   - Multiply the number by itself twice or use `**3`.
   - Print the result.

8. **Sum of three numbers**
   **Hint:**

   - Input three numbers.
   - Add them using `+`.
   - Print the sum.

9. **Average of three numbers**
   **Hint:**

   - Input three numbers.
   - Sum the numbers and divide by 3.
   - Print the result.

10. **Area of a rectangle**
    **Hint:**

    - Input length and breadth.
    - Multiply `length × breadth`.
    - Print the area.

11. **Perimeter of a rectangle**
    **Hint:**

    - Input length and breadth.
    - Formula: `2 × (length + breadth)`.
    - Print the perimeter.

12. **Area of a square**
    **Hint:**

    - Input the side.
    - Formula: `side ** 2`.
    - Print the area.

13. **Perimeter of a square**
    **Hint:**

    - Input the side.
    - Formula: `4 × side`.
    - Print the perimeter.

14. **Area of a circle**
    **Hint:**

    - Input radius.
    - Formula: `3.14 × radius ** 2`.
    - Print the area.

15. **Circumference of a circle**
    **Hint:**

    - Input radius.
    - Formula: `2 × 3.14 × radius`.
    - Print the circumference.

16. **Celsius to Fahrenheit**
    **Hint:**

    - Input Celsius.
    - Formula: `F = (C × 9/5) + 32`.
    - Print Fahrenheit.

17. **Fahrenheit to Celsius**
    **Hint:**

    - Input Fahrenheit.
    - Formula: `C = (F - 32) × 5/9`.
    - Print Celsius.

18. **Simple Interest**
    **Hint:**

    - Input Principal (P), Rate (R), Time (T).
    - Formula: `SI = (P × R × T)/100`.
    - Print the interest.

19. **Compound Interest**
    **Hint:**

    - Input Principal (P), Rate (R), Time (T).
    - Formula: `CI = P × (1 + R/100) ** T - P`.
    - Print the result.

20. **Swap two numbers**
    **Hint:**

    - Input two numbers `a` and `b`.
    - Use `a, b = b, a` to swap.
    - Print `a` and `b`.

21. **Area of a triangle**
    **Hint:**

    - Input base and height.
    - Formula: `0.5 × base × height`.
    - Print the area.

22. **Perimeter of a triangle**
    **Hint:**

    - Input three sides.
    - Formula: `side1 + side2 + side3`.
    - Print perimeter.

23. **Square root of a number**
    **Hint:**

    - Input a number.
    - Use `number ** 0.5`.
    - Print the result.

24. **Remainder when divided by 5**
    **Hint:**

    - Input a number.
    - Use `% 5` to get remainder.
    - Print it.

25. **Sum of digits of a 2-digit number**
    **Hint:**

    - Input number.
    - Find tens: `num // 10`, units: `num % 10`.
    - Sum them and print.

26. **Sum of digits of a 3-digit number**
    **Hint:**

    - Input number.
    - Hundreds: `num // 100`, tens: `(num // 10) % 10`, units: `num % 10`.
    - Sum them and print.

27. **Reverse a 2-digit number**
    **Hint:**

    - Input number.
    - Swap tens and units: `reversed = (num % 10) * 10 + (num // 10)`.
    - Print reversed number.

28. **Average of five numbers**
    **Hint:**

    - Input 5 numbers.
    - Sum and divide by 5.
    - Print average.

29. **Sum of numbers from 1 to n**
    **Hint:**

    - Input `n`.
    - Use formula: `sum = n * (n + 1) // 2`.
    - Print sum.

30. **Factorial of a number**
    **Hint:**

    - Input `n`.
    - Initialize `fact = 1`.
    - Multiply `fact` by numbers from 1 to n in a loop.
    - Print factorial.

31. **Area of parallelogram**
    **Hint:**

    - Input base and height.
    - Formula: `base × height`.
    - Print area.

32. **Perimeter of parallelogram**
    **Hint:**

    - Input two sides `a` and `b`.
    - Formula: `2 × (a + b)`.
    - Print perimeter.

33. **Area of trapezium**
    **Hint:**

    - Input `a`, `b` (parallel sides) and height `h`.
    - Formula: `0.5 × (a + b) × h`.
    - Print area.

34. **Perimeter of trapezium**
    **Hint:**

    - Input four sides.
    - Sum them.
    - Print perimeter.

35. **Area of rhombus**
    **Hint:**

    - Input diagonals `d1` and `d2`.
    - Formula: `0.5 × d1 × d2`.
    - Print area.

36. **Perimeter of rhombus**
    **Hint:**

    - Input side length.
    - Formula: `4 × side`.
    - Print perimeter.

37. **Double a number**
    **Hint:**

    - Input number.
    - Multiply by 2.
    - Print result.

38. **Triple a number**
    **Hint:**

    - Input number.
    - Multiply by 3.
    - Print result.

39. **Quotient and remainder of division**
    **Hint:**

    - Input two numbers.
    - Use `//` for quotient, `%` for remainder.
    - Print both.

40. **Sum of squares of two numbers**
    **Hint:**

    - Input two numbers.
    - Compute `a**2 + b**2`.
    - Print sum.

41. **Sum of cubes of two numbers**
    **Hint:**

    - Input two numbers.
    - Compute `a**3 + b**3`.
    - Print sum.

42. **Evaluate `(a + b) * c`**
    **Hint:**

    - Input `a`, `b`, `c`.
    - Compute `(a + b) * c`.
    - Print result.

43. **Evaluate `(a - b) / c`**
    **Hint:**

    - Input `a`, `b`, `c`.
    - Compute `(a - b) / c`.
    - Print result.

44. **Convert minutes into seconds**
    **Hint:**

    - Input minutes.
    - Multiply by 60.
    - Print seconds.

45. **Convert hours into minutes**
    **Hint:**

    - Input hours.
    - Multiply by 60.
    - Print minutes.

46. **Convert days into hours**
    **Hint:**

    - Input days.
    - Multiply by 24.
    - Print hours.

47. **Convert weeks into days**
    **Hint:**

    - Input weeks.
    - Multiply by 7.
    - Print days.

48. **Perimeter of a regular hexagon**
    **Hint:**

    - Input side.
    - Formula: `6 × side`.
    - Print perimeter.

49. **Area of a regular hexagon**
    **Hint:**

    - Input side.
    - Formula: `(3 * 1.732 / 2) * side ** 2` (√3 ≈ 1.732).
    - Print area.

50. **Total price with tax**
    **Hint:**

    - Input item price and tax %.
    - Formula: `total = price + (price * tax/100)`.
    - Print total.

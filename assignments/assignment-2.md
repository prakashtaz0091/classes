# 20 Practice Questions — `for` Loop

---

**1. Print numbers 1 to 10**

- Hints:

  1. Use `for` loop with range.
  2. Start from 1, end at 10.
  3. Print each number.

---

**2. Print even numbers from 1 to 20**

- Hints:

  1. Use `for` with step 2.
  2. Print only even numbers.

---

**3. Print odd numbers from 1 to 20**

- Hints:

  1. Loop through 1–20.
  2. Use `if num % 2 != 0`.
  3. Print odd numbers.

---

**4. Print multiplication table of a number (e.g., 5)**

- Hints:

  1. Take input number.
  2. Loop from 1 to 10.
  3. Multiply and print result.

---

**5. Print squares of numbers from 1 to 10**

- Hints:

  1. Loop 1–10.
  2. Multiply number × number.
  3. Print square.

---

**6. Print cubes of numbers from 1 to 5**

- Hints:

  1. Loop 1–5.
  2. Multiply number × number × number.
  3. Print cube.

---

**7. Calculate sum of numbers 1–100**

- Hints:

  1. Initialize sum = 0.
  2. Loop 1–100.
  3. Add each number to sum.
  4. Print sum.

---

**8. Calculate sum of even numbers from 1–50**

- Hints:

  1. Loop 1–50.
  2. Check even.
  3. Add to sum.

---

**9. Calculate factorial of a number**

- Hints:

  1. Input number.
  2. Initialize result = 1.
  3. Loop from 1–n.
  4. Multiply result each time.
  5. Print factorial.

---

**10. Print reverse numbers from 10 to 1**

- Hints:

  1. Use range with step -1.
  2. Start from 10, go down to 1.
  3. Print each number.

---

**11. Print first 10 multiples of 3**

- Hints:

  1. Loop from 1–10.
  2. Multiply by 3.
  3. Print.

---

**12. Print sum of digits of a number**

- Hints:

  1. Input number as string.
  2. Loop through each digit.
  3. Convert to int and add.
  4. Print sum.

---

**13. Count number of vowels in a word**

- Hints:

  1. Input a word.
  2. Loop each character.
  3. Check if vowel.
  4. Count.

---

**14. Print all characters of a string one by one**

- Hints:

  1. Input a string.
  2. Loop over string.
  3. Print each character.

---

**15. Print list of items with their serial number**

- Hints:

  1. Have a list (e.g., fruits).
  2. Use `for` loop with index.
  3. Print serial + item.

---

**16. Print stars in one line (**\*\*\* 5 times)\*\*

- Hints:

  1. Use `for` loop 5 times.
  2. Print `*` without newline.

**18. Calculate average of marks of 5 students**

**19. Print names of students from a list**

**20. Print each letter of your name in uppercase**

# 20 Practice Questions — `while` Loop

---

**1. Print numbers 1 to 10**

- Hints:

  1. Initialize `num = 1`.
  2. While `num <= 10`.
  3. Print and increment.

---

**2. Print numbers in reverse (10 to 1)**

- Hints:

  1. Start from 10.
  2. While greater than 0.
  3. Print and decrement.

---

**3. Print even numbers up to 20**

- Hints:

  1. Start from 2.
  2. While <= 20.
  3. Print and add 2.

---

**4. Print odd numbers up to 20**

- Hints:

  1. Start from 1.
  2. While <= 20.
  3. Print and add 2.

---

**5. Sum of first 50 natural numbers**

- Hints:

  1. Initialize sum = 0, num = 1.
  2. While <= 50.
  3. Add num to sum.
  4. Increment num.
  5. Print sum.

---

**6. Factorial of a number**

- Hints:

  1. Input n.
  2. Set result = 1.
  3. While <= n.
  4. Multiply result.
  5. Print factorial.

---

**7. Keep asking user for number until they enter 0**

- Hints:

  1. Use `while num != 0`.
  2. Input each time.
  3. Print number.

---

**8. Guess the secret number**

- Hints:

  1. Store secret.
  2. While guess != secret.
  3. Ask input again.
  4. Print success when correct.

---

**9. Print digits of a number one by one**

- Hints:

  1. Input number.
  2. While > 0.
  3. Use `% 10` to get digit.
  4. Use `// 10` to reduce number.

---

**10. Reverse a number**

- Hints:

  1. Input number.
  2. While > 0.
  3. Extract last digit.
  4. Add to reversed number.

---

**11. Check if a number is palindrome**

- Hints:

  1. Reverse number using while.
  2. Compare original and reversed.

---

**12. Print multiplication table using while**

- Hints:

  1. Input number.
  2. Start multiplier = 1.
  3. While <= 10.
  4. Print product.

---

**13. Find smallest divisor of a number (except 1)**

- Hints:

  1. Input number.
  2. Start divisor = 2.
  3. While <= number.
  4. Check if divisible.
  5. Stop at first divisor.

---

**14. Print powers of 2 less than 100**

- Hints:

  1. Start num = 1.
  2. While num < 100.
  3. Print num.
  4. Multiply num by 2.

---

**15. Sum of digits using while**

- Hints:

  1. Input number.
  2. While > 0.
  3. Get last digit.
  4. Add to sum.
  5. Reduce number.

---

**16. Keep asking password until correct one entered**

- Hints:

  1. Store correct password.
  2. While input != correct.
  3. Ask again.
  4. Print success when matched.

---

**17. ATM PIN system (3 attempts only)**

- Hints:

  1. Store correct PIN.
  2. Counter = 3.
  3. While attempts remain.
  4. Check input.
  5. Exit if correct or attempts over.

---

**18. Print characters of a string one by one using index**

- Hints:

  1. Input string.
  2. Start index = 0.
  3. While index < length.
  4. Print character.
  5. Increment index.

---

**19. Print running total until user types "stop"**

- Hints:

  1. total = 0.
  2. While input != "stop".
  3. Add number to total.
  4. Print final total.

---

**20. Countdown timer**

- Hints:

  1. Input number (say 10).
  2. While number > 0.
  3. Print number.
  4. Decrement until 0.

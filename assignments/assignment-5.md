# Practice Questions – Day’s Assignment

### **Tuple (3 Questions)**

1. Create a tuple of 5 numbers. Print the first and last elements.
   _Hint: Use indexing like `tuple[0]` and `tuple[-1]`._

2. Write a program to check if a given element exists inside a tuple.
   _Hint: Use the `in` keyword._

3. Convert a tuple into a list, add one new element, and convert it back to tuple.
   _Hint: Use `list()` and `tuple()` functions._

---

### **Set (3 Questions)**

4. Create two sets: one with numbers `{1,2,3,4,5}` and another `{4,5,6,7,8}`. Find their union and intersection.
   _Hint: Use `.union()` and `.intersection()`._

5. Create a set of fruits. Add a new fruit to it, then remove one fruit.
   _Hint: Use `.add()` and `.remove()`._

6. Write a program to find unique elements from a list using a set.
   _Hint: Convert list to set using `set(list_name)`._

---

### **List Slicing (3 Questions)**

7. Create a list of numbers from 1 to 10. Print:

   - First 5 numbers
   - Last 3 numbers
   - Every second number

   _Hint: Use `list[:5]`, `list[-3:]`, `list[::2]`._

8. Given a list `["a", "b", "c", "d", "e", "f"]`, print only `"b", "c", "d"`.
   _Hint: Use slicing with start and end index._

9. Reverse a list using slicing.
   _Hint: Use `list[::-1]`._

---

### **Packing and Unpacking (3 Questions)**

10. Pack three numbers into a tuple. Then unpack them into three variables `a, b, c` and print them.

11. Write a program to unpack the first two elements of a list into variables, and keep the rest in another variable.
    *Hint: Use `a, b, *rest = list_name`.\*

12. Write a function that takes three arguments using packing, and print them all.
    *Hint: Use `*args` in function parameter.\*

---

### **Dictionary (3 Questions)**

13. Create a dictionary with keys `"name"`, `"age"`, `"city"`. Print all the values.

14. Add a new key `"country"` to the dictionary and update `"age"` to a new value.

15. Write a program to loop through a dictionary and print all keys and values.
    _Hint: Use `.items()` inside a `for` loop._
